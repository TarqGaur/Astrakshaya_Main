Comment Fetcher AI - AI-Powered Social Media Comment Intelligence Platform

Transform social media comments into actionable insights with AI-powered audience intelligence, delivered instantly in the browser.

1. Project Overview

Comment Fetcher AI is a browser-based social media analytics platform that transforms large volumes of comments from YouTube and Instagram posts into actionable audience insights using artificial intelligence.

The system allows creators to paste a post URL and instantly receive deep analysis of audience sentiment, discussion trends, spam activity, and viewer personas — all directly inside the browser.

Unlike traditional analytics tools that require backend servers or multiple API calls, this platform performs full comment analysis in a single AI request, making the system extremely fast and cost-efficient.

Key design goals:

  • Zero backend infrastructure
  • Single HTML application
  • AI-powered analytics in seconds
  • Creator-focused insights

Everything runs directly in the browser using Vanilla JavaScript and modern APIs.

2. Key Features

1. Multi-Platform Comment Fetching

The system automatically detects the platform from the URL and fetches comments accordingly.

Supported platforms:

  • YouTube videos
  • Instagram posts

Data includes:

  • comment text
  • username
  • timestamps
  • likes
  • replies
  • profile information

Up to hundreds of comments can be collected in seconds.

2. Instant Post Analytics Dashboard

Once comments are fetched, the system generates a full analytics dashboard including 10+ interactive sections.

The dashboard visualizes audience behaviour using:

  • charts
  • tags
  • sentiment bars
  • persona cards
  • discussion threads

All sections are rendered dynamically inside the browser.

3. AI-Powered Sentiment Analysis

The platform analyzes audience mood by classifying comments into:

  • Positive
  • Neutral
  • Negative

Creators can quickly understand how their audience feels about the content.

The system also generates:

  • overall emotional tone
  • key insight explaining the sentiment

4. Automated Audience Summary

Instead of manually reading hundreds of comments, the system generates a concise summary of viewer reactions.

The AI identifies:

  • common praise
  • recurring complaints
  • overall emotional tone
  • suggested creator improvements

This allows creators to quickly understand what worked and what didn't in their content.

5. Topic & Discussion Identification

The platform detects the main topics viewers are discussing in the comments.

Output includes:

  • trending discussion topics
  • descriptive headline
  • keyword tags with frequency

Creators can discover what aspect of their video or post is driving conversation.

6. Audience Personality Insights

The system groups commenters into audience profiles such as:

  • supportive fans
  • critics
  • curious learners
  • technical discussers

Each profile includes:

  • percentage share
  • typical opinions
  • behaviour patterns

This helps creators understand their community composition.

7. Country Opinion Map

Using language patterns and cultural cues, the AI estimates which countries comments originate from.

For each country the system shows:

  • positive comments
  • neutral comments
  • negative comments

Creators can see how different regions react to their content.

8. Hate Source Detection

A visual pie chart highlights which regions contribute the most negative comments.

This helps creators:

  • identify targeted trolling
  • understand regional audience reactions
  • track sentiment by geography

9. Spam & Bot Detection

The platform analyzes comment patterns to detect:

  • spam messages
  • bot-like behaviour
  • suspicious promotional comments

A visual chart shows the composition of:

  • legitimate users
  • spam accounts
  • automated bots

Creators can quickly see how authentic their engagement is.

10. Top Discussion Thread Finder

The system automatically detects the most active conversation in the comments section.

Selection algorithm:

  • highest reply count
  • highest likes (tie breaker)

The entire thread is displayed so creators can see the most engaging audience conversation.

11. One-Word Audience Insight

The AI generates a quick high-level reaction summary including:

  • overall sentiment
  • biggest praise
  • biggest complaint
  • dominant emotion
  • recommended creator action

This acts as a quick decision snapshot for creators.

12. AI Audience Personas

The platform builds realistic audience personas such as:

  • tech enthusiasts
  • casual viewers
  • critics
  • learners

Each persona includes:

  • age group estimate
  • interests
  • mood
  • engagement style

This helps creators better understand who their audience actually is.

13. Engagement Prediction

Using comment sentiment and behaviour patterns, the system predicts future engagement trends including:

  • expected likes vs dislikes
  • overall approval rating
  • prediction confidence

Creators can estimate how their content will perform over time.

14. AI Comment Q&A Assistant

A built-in AI chat system allows creators to ask questions like:

  • "What are viewers complaining about most?"
  • "What do Indian viewers think about this video?"
  • "Which comment trend is growing fastest?"

The AI answers using the actual comment dataset.
